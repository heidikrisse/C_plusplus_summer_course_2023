# Build Project

1. Navigate to the project root directory

2. Create a build directory and navigate to it:

```shell
mkdir build
cd build
```

3. Run CMake and build the project:

```shell
cmake ..
make
```

# Run Project

```shell
./Memerror2
```

# Run Unit Tests

```shell 
../tests/unit_tests_memerror2
``` 
